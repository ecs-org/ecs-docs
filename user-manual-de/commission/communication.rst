===========
Nachrichten
===========

Sie können jeder systeminternen Person eine Nachricht schreiben. Unter *Beteiligte Parteien* im Tab *Generell* ist es Ihnen außerdem möglich, direkt eine Nachricht an eine beteiligte Person der Studie zu schreiben. Wollen Sie eine Nachricht abgeben, können Sie das in der Ansicht des Nachrichtenstrangs tun. Informieren Sie gegebenenfalls den Empfänger des neuen Nachrichtenstrangs.
