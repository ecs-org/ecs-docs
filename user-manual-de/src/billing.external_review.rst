=========================
Gutachter/innen Bezahlung
=========================

Die Filter ermöglichen eine Auswahl der Gutachter/innen, die bezahlt werden sollen. Klicken Sie anschließend auf *Daten absenden*, um zur Gebührenauflistung zu gelangen. Die Auflistung finden Sie als Excel-Datei unter dem Link *Detaillierte Auflistung*. Sie können die Datei nun an die Finanzabteilung schicken.
