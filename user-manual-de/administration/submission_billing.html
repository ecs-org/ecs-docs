Gebühren¶

Die Filter ermöglichen eine Auswahl der studienbezogenen Rechnungen. Treffen Sie eine Auswahl und klicken Sie auf Daten absenden, um zur Rechnungsstellung zu gelangen. Die Auflistung finden Sie als Excel-Datei unter dem Link Detaillierte Auflistung. Sie können die Datei nun weiter an die Finanzabteilung schicken.
